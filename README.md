## Run Locally

Clone the project

```bash
  git clone https://github.com/Lokesh01/Shopping-cart-useContext
```

Go to the project directory

```bash
  cd shopping_cart_ts
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run start
```

Start the Client
